\doxysubsection{Detailed Description}
Permet de stocker une liste d\textquotesingle{}éléments de façon dynamique. 

\begin{DoxyRemark}{Remarks}
Cette classe a été créé avec comme idée de ne stocker que des valeurs \char`\"{}simples\char`\"{}, c\textquotesingle{}est-\/à-\/dire des types primitifs ou des structures. Le stockage d\textquotesingle{}objets complexes est possible mais l\textquotesingle{}ajout de ceux-\/ci dans une liste sera donc plus long. 
\end{DoxyRemark}
